# Droppy Raycast Extension

Add files to Droppy directly from Raycast!

## Installation

1. Open Raycast
2. Go to Extensions â†’ Import Extension
3. Select this folder (`Raycast/`)
4. Or run `npm install && npm run dev` in this directory

## Commands

- **Add to Droppy Shelf** - Add selected Finder files to the notch shelf
- **Add to Droppy Basket** - Add selected Finder files to the floating basket

## Requirements

- [Droppy](https://github.com/iordv/Droppy) v4.9+
- [Raycast](https://raycast.com) (free)
