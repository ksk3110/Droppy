## What's Droppy?

**The native productivity layer macOS is missing.**

Stop juggling single-purpose utilities. Droppy brings your clipboard history, file shelf, screenshot tools, and system HUDs together in one native interfaceâ€”all living inside your notch.

**No notch?** Droppy adds a Dynamic Island-style pill to any Mac.

### The basics:

- **Shelf & Basket** â€” Drag files to the notch to stash them. Jiggle your mouse while dragging to summon a floating basket anywhere on screen.
- **Clipboard History** â€” Full history with search, favorites, OCR text extraction, and drag-out.
- **Native HUDs** â€” Clean replacements for volume, brightness, battery, caps lock, and more.
- **Media Controls** â€” Album art, playback controls, and a seek slider right in the notch.

## Install

### Homebrew (recommended)
```bash
brew install --cask iordv/tap/droppy
```

### Manual Download
1. Download [**Droppy.dmg**](https://github.com/iordv/Droppy/releases/latest)
2. Clear quarantine (required for unsigned apps):
   ```bash
   xattr -rd com.apple.quarantine ~/Downloads/Droppy-*.dmg
   ```
3. Open the DMG and drag Droppy to Applications

> ðŸ’¡ Homebrew handles quarantine automatically and makes updates easy.
